# SunMoon
Note taking and to-do list app

### About :

The idea of the project is to combine the functionality of note-taking and to-do list applications

---

### Features :

- You can create note or task (with sub-tasks)
- Each task can be added with or without a date. These 2 types of tasks will be separated and presented in different tabs but you can remove the date from scheduled tasks and add it to unscheduled
- Overdue tasks are presented in a separate tab and you can see how late the task is
- You can add a reminder to a task and specify the exact time of this reminder
- You can add a group for notes and sort them by it
- You can archive a note
- You can mark a group of notes as invisible and it will not be shown on the notes screen
- You can search tasks and notes by title and description
- All deleted tasks and notes are first placed in the trash
- You can change the app theme date&time format and language
- The application does not require an internet connection and user permissions

---

### Architecture & Tools :

- MVVM Clean architecture
- Hilt
- Coroutines
- Room
- DataStore
